2. Admin Flow (The Setup & Management)
The Admin sets the stage before anyone else can act.

Scenario A: Setting up the Academic Structure

Login → Go to Academic Module.

Create Department (e.g., Computer Science).

Create Course & Batch (e.g., B.Tech 2024-2028).

Create Subjects → Map them to the specific Course/Semester.

Scenario B: User Onboarding

Go to User Management.

Select Role (Teacher or Student).

Create User (Input Name, Email, etc.).

Generate Credentials (System auto-generates ID/Pass).

Assign Context:

If Student: Assign to a specific Batch/Section.

If Teacher: Assign to specific Subjects/Classes.

Scenario C: Creating the Time-Table

Go to Time-Table Module.

Select Class + Day.

Add Slot: Select Time + Subject + Teacher + Room.

Save (System checks for conflicts).
