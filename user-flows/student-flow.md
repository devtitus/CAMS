4. Student Flow (Consumption)
Students primarily view data.

Scenario A: Checking Daily Routine

Login → View Student Dashboard.

Immediate View: See "Today's Time-Table" widget.

Action: Check which room to go to for the next class.

Scenario B: downloading Study Material

Go to Notes Module.

Filter: Select Subject (e.g., Mathematics).

View list of uploaded notes.

Click Download on the desired file.

Scenario C: Checking Results

Go to Assessment Module → Click Results.

Select Semester.

View Report Card (Subject-wise breakdown of Internal/External marks).
