3. Teacher Flow (Daily Operations)
Teachers use the system to manage their day-to-day tasks.

Scenario A: Marking Attendance (The Happy Path)

Login → View Teacher Dashboard.

Check Schedule: See current/upcoming class (e.g., "10:00 AM - Java").

Click "Mark Attendance" on the active class card.

System fetches student list for that specific batch.

Mark Status: Toggle "Present/Absent" (Default can be "Present" to save clicks).

Submit.

Scenario B: Uploading Notes

Go to Notes Module.

Click "Upload Note".

Select Subject (Dropdown shows only assigned subjects) → Enter Topic Name.

Attach File (PDF/PPT).

Publish.

Scenario C: Entering Marks

Go to Assessment Module.

Select Exam Type (e.g., Mid-Term) → Select Subject.

Enter Marks next to each student's name in the list.

Lock & Submit (Sends to Admin for final result generation).
